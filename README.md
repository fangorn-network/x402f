# x402f 

## Setup

First setup environment variables by running `cp .env.local .env` and filling in the details.

## Start the Facilitator

`npm run facilitator`

## start the server

`npm run server`

## run the client example

install tsx  with `npm install -D tsx`

Then run the example with

`npm run client`
